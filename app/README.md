# OWI Field App (`/app`)

Collector PWA (ADR-6): photo capture, QR/GPS bin identification, fill tap, offline sync queue, route view.

Stack: Vite + React + TypeScript, Workbox service worker, IndexedDB queue. First task: the Phase 0 spike — camera + offline + GPS on an Android 10 / 2 GB phone validates the PWA decision before anything else is built.
