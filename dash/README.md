# OWI Dashboard (`/dash`)

Ops dashboard for coordinators and directors: composition views, bin health, collect-today list, review queue, reports.

Stack: React + Vite + TypeScript + **Apache ECharts** (the only chart library, ADR-8). UI rules from [docs/10-system-spec.md](../docs/10-system-spec.md): flat, no gradients, responsive, fast, EN+SW i18n from the first screen.
